# 학습 자료 요약

이 저장소는 개인 학습 정리 PDF와 강의용 자료구조 PDF를 참고하여 GitHub 업로드용으로 재구성한 학습 기록입니다.

## 개인 정리 PDF에서 확인한 내용

개인 정리 PDF에는 다음 내용이 포함되어 있었습니다.

- 프로그램 = 알고리즘 + 자료구조
- 알고리즘 조건
- 복잡도 분석과 Big-O
- 선형 자료구조와 비선형 자료구조
- 배열 구조와 연결 구조
- 리스트, 스택, 큐, 우선순위 큐
- 선택 정렬, 삽입 정렬, 버블 정렬
- 집합 연산
- 순차 탐색, 이진 탐색, 보간 탐색
- 해싱과 충돌 처리
- 트리, 이진 트리, 순회
- 힙, 이진 탐색 트리, 그래프, MST

## 강의용 자료구조 PDF를 참고하여 보강한 내용

강의용 자료에서는 다음 개념을 보강하는 데 참고했습니다.

- ADT의 의미와 인터페이스 중심 사고
- 시간 복잡도, 공간 복잡도, 점근적 표기법
- Big-O, Big-Ω, Big-Θ
- 리스트/집합 ADT
- 스택, 큐, 덱, 우선순위 큐의 활용
- 연결 리스트의 노드 구조
- 정렬 알고리즘 분류
- 탐색, 해싱, 맵 구현 방식
- 이진 트리, 힙, BST, AVL
- 그래프 표현, DFS, BFS, 연결 성분, 위상 정렬
- MST, Kruskal, Prim, 최단 경로 알고리즘

## 재구성 방식

강의자료의 문장을 그대로 옮기지 않고, 개인 학습 기록으로 보이도록 다음 방식으로 재구성했습니다.

- GitHub 첫 화면용 README 작성
- 주제별 Markdown 파일 분리
- 표와 코드 예제 추가
- 직접 구현할 수 있는 Python 예제 파일 추가
- 학습 체크리스트 추가
