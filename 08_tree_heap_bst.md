# 08. 트리, 힙, 이진 탐색 트리

## 1. 트리란?

트리(Tree)는 계층적인 관계를 표현하는 비선형 자료구조입니다.

대표적인 예시는 다음과 같습니다.

- 회사 조직도
- 가계도
- 컴퓨터 폴더 구조
- 의사결정 구조
- 탐색 트리
- 우선순위 큐 구현을 위한 힙

## 2. 트리 용어

| 용어 | 의미 |
|---|---|
| 노드 | 데이터를 저장하는 기본 단위 |
| 루트 | 가장 위에 있는 노드 |
| 부모 노드 | 어떤 노드의 바로 위 노드 |
| 자식 노드 | 어떤 노드의 바로 아래 노드 |
| 단말 노드 | 자식이 없는 노드 |
| 간선 | 노드와 노드를 연결하는 선 |
| 높이 | 루트에서 가장 깊은 노드까지의 거리 |

## 3. 이진 트리

이진 트리는 각 노드가 최대 2개의 자식 노드를 갖는 트리입니다.

```text
        A
      /   \
     B     C
    / \
   D   E
```

## 4. 이진 트리 표현 방법

| 방법 | 특징 |
|---|---|
| 배열 표현 | 완전 이진 트리에 적합, 구현 간단 |
| 링크 표현 | 노드와 링크 사용, 일반적인 트리에 유연함 |

## 5. 이진 트리 순회

순회는 트리의 모든 노드를 한 번씩 방문하는 과정입니다.

| 순회 방식 | 방문 순서 | 활용 예 |
|---|---|---|
| 전위 순회 | 루트 → 왼쪽 → 오른쪽 | 구조 출력 |
| 중위 순회 | 왼쪽 → 루트 → 오른쪽 | 이진 탐색 트리 정렬 출력 |
| 후위 순회 | 왼쪽 → 오른쪽 → 루트 | 폴더 용량 계산 |
| 레벨 순회 | 레벨 순서대로 방문 | 큐 사용 |

```python
def preorder(node):
    if node is None:
        return
    print(node.data)
    preorder(node.left)
    preorder(node.right)
```

## 6. 힙

힙(Heap)은 완전 이진 트리를 기반으로 하는 자료구조입니다.  
최대 힙은 부모 노드가 자식 노드보다 크거나 같고, 최소 힙은 부모 노드가 자식 노드보다 작거나 같습니다.

Python에서는 `heapq`가 최소 힙을 제공합니다.

```python
import heapq

heap = []
heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappush(heap, 2)

print(heapq.heappop(heap))  # 1
```

## 7. 힙의 활용

- 우선순위 큐
- 힙 정렬
- 다익스트라 알고리즘
- 허프만 코딩

## 8. 이진 탐색 트리

이진 탐색 트리(Binary Search Tree, BST)는 탐색을 효율적으로 하기 위한 이진 트리입니다.

규칙은 다음과 같습니다.

```text
왼쪽 서브트리의 모든 키 < 루트 키 < 오른쪽 서브트리의 모든 키
```

## 9. BST 탐색

```python
def search_bst(root, key):
    if root is None:
        return None

    if key == root.key:
        return root
    elif key < root.key:
        return search_bst(root.left, key)
    else:
        return search_bst(root.right, key)
```

평균적으로 `O(log n)`이지만, 트리가 한쪽으로 치우치면 `O(n)`이 될 수 있습니다.

## 10. 균형 이진 탐색 트리

균형 이진 탐색 트리는 트리의 높이를 일정하게 유지하여 탐색, 삽입, 삭제 성능을 안정적으로 유지합니다.

대표적인 예시는 AVL 트리입니다.

AVL 트리는 삽입이나 삭제 후 균형이 깨지면 회전을 통해 다시 균형을 맞춥니다.

| 회전 | 상황 |
|---|---|
| LL 회전 | 왼쪽-왼쪽으로 치우침 |
| RR 회전 | 오른쪽-오른쪽으로 치우침 |
| LR 회전 | 왼쪽-오른쪽으로 치우침 |
| RL 회전 | 오른쪽-왼쪽으로 치우침 |

## 11. 핵심 정리

- 트리는 계층 구조 표현에 적합하다.
- 이진 트리는 각 노드가 최대 2개의 자식을 갖는다.
- 전위, 중위, 후위, 레벨 순회 방식이 있다.
- 힙은 우선순위 큐 구현에 활용된다.
- BST는 탐색을 빠르게 하기 위한 이진 트리이다.
- 균형 트리는 한쪽으로 치우친 BST의 성능 저하를 보완한다.
